## ⚡ Inicio Rápido

```bash
# Navegar al proyecto
cd arcade-modern

# Instalar dependencias (si no lo has hecho)
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build
```
